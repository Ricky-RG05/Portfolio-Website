Liquor Store Financial
Analysis Tool in Python

This project analyzes real-world sales data from a liquor store located in Madrid, Spain.

Using Python, I developed an interactive tool that provides financial insights and visualizations through a user-friendly menu. The tool includes features like descriptive statistics, bar graphs, scatter plots, and pie charts, helping uncover trends and opportunities for business growth.

The project also includes a detailed 10+ page analysis essay with actionable recommendations for improving the store's business model.

The analysis was performed on a subset of the original database (50 rows) to meet project requirements. The code is adaptable to the full dataset for future scalability.
